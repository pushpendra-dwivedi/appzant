MySQL Database Upgradation and Migration

Duration: Jan 2022 to June 2023
Technology: mysql

  • - Get this entire process approved by Oracle and after that implemented in approximately 143 divisions with minimum downtime.
  • - After upgradation, migration and configuring replication validation of migrated data done successfully with different approaches.
  • - Prepared hardening process for same and implemented on all servers.
